ACT.MPTimer
===========

概要
-------------
ゲーム内の3秒ごとのMP回復をプログレスバーで表示します
  
  
使い方
--------------
ACT.MPTimer.dll  
ACT.MPTimer.dll.config  
をACTのインストールディレクトリにコピーします  
その後、プラグインとしてACT.MPTimer.dllを追加してください  
  
ゲーム内で自分のMPが減って自然回復すると回復タイミングが同期されます
